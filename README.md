## 安裝
前端:
```bash
# At final/
cd frontend
yarn install
```
後端:
```bash
# At final/
cd backend
yarn install
```
並在`.env.defaults`中的`MONGO_URL`中填入Mongo Atlas DB的URL
## 執行
```bash
yarn frontend
yarn backend
```
